# 🚀 Space Invaders

A beginner-friendly recreation of the classic **Space Invaders** arcade game built using **Python** and **Pygame**. 

## Features

- Player-controlled spaceship
- Enemy movement with multiple enemies
- Bullet shooting mechanics
- Collision detection
- Score tracking
- Sound effects and background music
- Game Over screen
- Custom images and game assets

## Tech Stack

- Python
- Pygame

## Controls

- **← / A** – Move Left
- **→ / D** – Move Right
- **Spacebar** – Shoot
